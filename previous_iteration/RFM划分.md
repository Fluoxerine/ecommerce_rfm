# RFM划分





# RFM 19 类 = 3 (R) × 2 (F) × 3 (M) + 1 (VIP)

**R 3 档**：1-60 / 61-180 / 181+（活跃 / 唤醒 / 流失）

**F 2 档**：1 次 / 2+ 次（一次性 / 复购）

**M 3 档**：0~25% / 25%~75% / 75%~ 上边缘（低 / 中 / 高）

**VIP 1 档**：M > 箱线图上边缘（单独顶级）



## R 3 档

1. 1~60 天 = 活跃
2. 61~180 天 = 沉睡 / 唤醒
3. 181+ 天 = 流失



## F 2 档

1. F=1 = 一次性用户（最需要提升复购）
2. F≥2 = 复购用户（最有价值）



## M 3 档 + VIP（解决长尾）

1. 0~25% = 低价值

2. 25%~75% = 中价值

3. 75% ~ 箱线图上边缘 = 高价值

4. > 箱线图上边缘 = VIP（单独 1 类）
