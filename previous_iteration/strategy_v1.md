# RFM客群差异化运营策略文档

## 1. 项目背景

本文档基于RFM分群分析结果，为**7大运营组**制定差异化运营策略。通过分析不同客群的用户行为特征、品类偏好和品牌偏好，制定针对性的运营方案，提升用户活跃度、转化率和客单价。

**方法论说明**：本项目采用自下而上的分析方法。首先基于实际数据分布确定RFM阈值（R=60/180天, M=P25/P50, VIP=箱线图上边缘），然后计算19个RFM基础分群（3×2×3+1VIP），最后根据运营需求合并为7个可执行差异化策略的运营组。

### 数据来源
- 分析日期：订单最晚日期的后一天
- 分析维度：RFM客群 x 品类 x 品牌
- 数据库：ecommerce_rfm
- 汇总表：dws_customer_category_pref

### 阈值确定（按RFM划分.md文档规格）
| 维度 | 档位1(高/活跃/复购) | 档位2(中/唤醒/一次性) | 档位3(低/流失) |
|------|-----|-----|-----|
| R（天） | <=60 (活跃) | 61-180 (唤醒) | >180 (流失) |
| F（次） | >=2 (复购) | 1 (一次性) | - |
| M（元） | >P75 (高) | P25-P75 (中) | <=P25 (低) |
| VIP | M > 箱线图上边缘 (Q3+1.5*IQR) | - | - |

## 2. 分群标准

### RFM模型说明
RFM模型是衡量用户价值和用户创利能力的经典方法，通过三个维度评估用户状态：
- **R（Recency）**：最近一次消费时间，距离越近价值越高
- **F（Frequency）**：消费频率，频繁消费的用户价值更高
- **M（Monetary）**：消费金额，消费金额高的用户更有价值

**运营策略**

1. **专属VIP服务**
   - 开通1对1专属客服通道
   - 生日/节日专属礼品赠送
   - 优先体验新品特权

2. **高端产品推荐**
   - 推荐高客单价商品
   - 推送高端品牌新品

- 月均消费频次：保持2次以上
- 客单价提升：目标提升20%
- 留存率：100%

## 4. 执行优先级

### 优先级矩阵

| 优先级 | 运营组 | 行动理由 | 占比估算 |
|--------|--------|----------|----------|
| P0 | 至尊VIP | 核心资产，必须维护 | ~5% |
| P1 | 核心高价值忠诚用户 | 高价值，需保持 | ~10% |
| P1 | 活跃复购潜力用户 | 高潜力，培养价值高 | ~15% |
| P2 | 活跃一次性新客 | 未来主力，需培育 | ~20% |

| P4 | 流失低价值低效用户 | 量大，低成本维护 | ~25% |
